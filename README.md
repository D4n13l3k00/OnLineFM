# OnLineFM

### Простое онлайн радио на C# для Wndows на базе Windows Media Player

![Screenshot](app.png)


### Фишки:
- Список любимых станций
- Таймер сна
- Возможность воспроизводить свои станции
- Возможность воспроизводить YouTube видео/стримы
